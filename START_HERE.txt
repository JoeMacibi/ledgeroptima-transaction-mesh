TROUBLESHOOTING

  Dashboard won't load?
  → Wait 10 seconds for database initialization
  → Check http://localhost:8000 in browser
  → Review docker-compose logs: docker-compose logs -f api

  No data showing?
  → Wait 30 seconds for synthetic data generation
  → Refresh browser with Ctrl+R (or Cmd+R on Mac)
  → Check console for errors with F12

  Port 8000 already in use?
  → Change docker-compose.yml port mapping
  → Or run on different port: uvicorn app.main:app --port 8001

  More help?
  → See QUICKSTART.md troubleshooting section
  → See DEPLOYMENT.md for detailed setup
  → Check README.md for features

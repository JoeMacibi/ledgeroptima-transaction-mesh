TROUBLESHOOTING

  Dashboard Not Loading?
  ──────────────────────
  1. docker-compose ps         (verify containers running)
  2. docker-compose logs api   (check for errors)
  3. Wait 10 seconds           (database initialization)
  4. Refresh browser           (Ctrl+R or Cmd+R)

  No Data Appearing?
  ──────────────────
  1. Wait 30 seconds           (data generation)
  2. Refresh browser           (clear cache)
  3. Check API endpoint        (http://localhost:8000/api/system/health)
  4. Review logs               (docker-compose logs -f api)

  Port 8000 Already in Use?
  ─────────────────────────
  1. Edit docker-compose.yml
  2. Change port: "8001:8000"
  3. docker-compose up -d
  4. Access: http://localhost:8001

  Database Connection Error?
  ──────────────────────────
  1. docker-compose down
  2. docker-compose up -d
  3. Wait 15 seconds for database
  4. Check docker-compose logs
